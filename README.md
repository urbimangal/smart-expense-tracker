# Smart Expense Tracker

## Description
Smart Expense Tracker is a C++ based application designed to help users record and manage their daily expenses. The project focuses on applying core programming and data structure concepts to build a simple and practical expense management system.

## Features
- Add daily expenses with category and description
- View all recorded expenses
- Calculate total expenses automatically
- Category-wise and monthly expense summary
- Simple and easy-to-use interface

## Technologies Used
- C++
- STL (vector, map)
- File handling

## Purpose
This project was developed to practice Data Structures, Object-Oriented Programming, and logical problem-solving through a real-world use case.

## How to Run
1. Clone the repository  
   git clone https://github.com/urbimangal/smart-expense-tracker

2. Compile the program  
   g++ main.cpp -o expense_tracker

3. Run the application  
   ./expense_tracker
